# Log Prompt - 102022400256_Monitoring-BBM-Servis-Service

Dokumen ini berisi narasi log *prompt* (instruksi) yang digunakan secara khusus untuk menyelesaikan pengembangan *backend API service* seputar Operasional dan Monitoring BBM (`102022400256_Monitoring-BBM-Servis-Service`). Layanan ini berfokus murni pada penyediaan API internal.

### 1. Inisialisasi Proyek Backend
"Tolong buatkan struktur proyek Laravel baru khusus untuk layanan Monitoring BBM dan Servis Kendaraan. Setelah itu, konfigurasikan akses koneksi databasenya di file `.env`."

### 2. Pembuatan Skema Database
"Bantu saya membuat file *migration* dan *model* untuk tabel operasional (Maintenance/BBM). Kolom yang dibutuhkan antara lain: id, schedule_id, vehicle_id, fuel_limit, last_service_date, dan operational_coupon."

### 3. Pembuatan API Endpoint
"Tolong buatkan *Controller* sederhana untuk mengelola data operasional tersebut. Buatkan rute API standar untuk melayani fungsionalitas GET (menampilkan daftar data laporan BBM), POST (menyimpan data input operasional baru), dan DELETE (menghapus data). Arahkan rute *endpoint*-nya ke `/api/v1/maintenance`."
